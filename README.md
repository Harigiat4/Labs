## 🌟 Program Overview

This repository archives the coursework and final project developed during the **Northwestern University Center for Talent Development (CTD) Summer Program**. 

This learning opportunity was made possible through the generous sponsorship of the **Mawhiba Safeer** program during my high school studies, allowing me to build a strong foundation in C++, microcontroller firmware, and IoT system architecture with **Windy City Lab**.

---

## 📂 Repository Structure

The curriculum was structured progressively, starting from basic hardware interactions to complex, cloud-connected multithreaded applications.

### 🔬 1. Foundation Labs (`lab1` - `lab19`)
A series of 19 progressive labs focusing on:
- Reading digital and analog hardware sensors.
- Pulse Width Modulation (PWM) and custom component control.
- Wi-Fi connectivity and Cloud data transmission.
- Core communication protocols (I2C, SPI).

### 🚀 2. Capstone Project (`capstonedemo/`)
The culmination of the summer program. It is an advanced IoT application built on a Particle device that integrates with the **Blynk** mobile application over Wi-Fi. 
- Features a **Virtual Joystick** for real-time `xy` coordinate control.
- Dynamic **RGB LED Sequencing**.
- Hardware **Multithreading** for seamless cloud-to-device communication.

👉 **[View the Capstone Project Details here!](./capstonedemo/README.md)**
